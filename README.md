# 人生管理系统

一个简洁温暖的人生管理网页，帮助你规划和管理生活的各个维度。

## 功能特点

- 🏠 **首页看板**：6张项目卡片一览无余
- 🔄 **自由拖拽**：卡片位置可以随意调整
- ➕ **自由添加**：支持添加自定义项目
- 🗑️ **自由删除**：不需要的项目可以删除
- 🔗 **财务模块**：投资（跳转交易系统）+ 开支

## 默认项目

| 项目 | 说明 |
|------|------|
| 💪 身体 | 健康和运动管理 |
| 💰 财务 | 投资 + 开支管理 |
| 📄 论文 | 学术研究和论文进度 |
| 💼 职业发展 | 职业规划和技能提升 |
| 📚 积累 | 知识、经验、资源积累 |
| 🧠 心理 | 心理健康和情绪管理 |

## 技术栈

- React + Vite
- Tailwind CSS（暖色系）
- @dnd-kit（拖拽排序）
- React Router

## 启动

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3001

## 交易系统集成

点击财务 → 投资，可以直接跳转到交易系统（http://localhost:3000）
